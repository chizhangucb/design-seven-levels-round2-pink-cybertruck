Level 3 of 7

Design skills

Install packaged design expertise so the model knows what good looks like — then it stops eating crayons and starts playing the piano.

The technique

Skills are prepackaged instruction sets written by people who obsess about design: UI styles, color palettes, font pairings, UX rules. Point Claude at a skill repo (ui-ux-pro-max, power-design, shadcn) and it internalizes the rulebook before writing a line of CSS.

This level combined two skill repos with a real product photo of the truck — and unlike round 1, both models got the identical prompt (round 1's inputs were asymmetric; disclosed deviation). The results grew to 2.4–2.9× the size of level 2. That's where the design system went.

The prompts that did it

Invoke every relevant design skill
Hey there. I'd like to use all the best skills and the design principles to build me a beautiful website selling pink Cybertrucks. Make sure that's using all relevant skills that I have given you for building beautiful design @reference-real-cybertruck.png https://github.com/ItsssssJack/power-design.git https://github.com/nextlevelbuilder/ui-ux-pro-max-skill.git Btw, do NOT use superpowers plugin in this session. keep pink-cybertruck-web-level-2.html untouched. name the file as pink-cybertruck-web-level-3.html

Fable shipped a 46.8 KB page (2.4× level 2) and ran a self-QA pass that caught and fixed a JS-gating bug — while honoring the superpowers exclusion. Opus 4.8 shipped 50.7 KB (2.9×), invented a fictional "CYBR" brand complete with its own disclaimer, and generated an extra hero.png asset alongside the page.

Cost & time

What changed vs level 2

  • A coherent design system appears: consistent spacing scale, type hierarchy, real color palette.
  • File size jumps 2.4–2.9× — the craft is in the CSS.
  • The models start catching their own design bugs when allowed to verify.

What this level taught

  • Skills > instructions: packaged expertise compounds with raw model ability.
  • Identical inputs make the A/B clean — the round-1 asymmetry is fixed this round.
  • Left to fill a brand vacuum, models invent one: Opus's fictional "CYBR" arrived unprompted.
