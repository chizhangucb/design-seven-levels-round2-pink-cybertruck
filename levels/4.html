Level 4 of 7

UI snipping

Don't re-engineer the fireplace — copy it. Production-grade components from 21st.dev, Magic UI, or CodePen drop straight in.

The technique

Component libraries publish battle-tested interactive pieces. Copy the code and ask Claude to integrate. This round: DottedSurface, a 21st.dev-style React + three.js animated dot-wave — always on, no trigger — swapped in for round 1's balloons (disclosed deviation; an ambient industrial particle field suits a Cybertruck better).

The full component source — React, next-themes, Tailwind utilities, three.js imports — was pasted into the message ahead of the prompt below, identically for both models. Adapting that stack into a static vanilla-HTML page is the level's actual work.

Note: the source video does image/video at level 4 and UI snipping at level 5 — these runs keep round 1's swapped order.

The prompts that did it

Integrate the snipped component (after the pasted source)
Hey there. I'd like you to integrate this dotted surface component at the bottom of the page. It should always be there, no trigger needed. Make sure you keep the current index HTML file. keep pink-cybertruck-web-level-3.html untouched. Name it as pink-cybertruck-web-level-4.html

Preceded by the component's full source pasted into the same message. Both models kept the component's full-viewport fixed inset-0 wrapper — exactly what the source does, and exactly the failure the run doc predicted — so the scripted rescope follow-up fired for both.

Rescope the surface (follow-up, both models)
The dotted surface should show at the bottom part of the page, not cover the whole page.

Both rescoped the canvas into a bottom-of-page section. Opus 4.8 caught the 0–255 vertex-color clamp bug in the source (three.js reads that attribute as 0–1), picked the classic three.js build for file:// compatibility, masked the top edge to hide the seam, and pauses the loop on hidden tabs. Fable pulled three.js from a CDN, verified the dots visible with Playwright, and added a prefers-reduced-motion static frame unprompted.

Cost & time

What changed vs level 3

  • Production-quality ambient motion appears with near-zero design risk.
  • Both models tripped over the same wrapper — a React component's layout assumptions survive translation.
  • The divergence axis moved: not inline-vs-CDN this time, but source-bug spotting (Opus) vs accessibility instinct (Fable).

What this level taught

  • The internet is a parts bin; snipping beats reinventing.
  • Say where the component goes — and expect to say it twice when the source hardcodes a full-viewport wrapper.
  • Models can find real upstream bugs during integration: Opus flagged the vertex-color clamp the docs had listed as a watch-for.
