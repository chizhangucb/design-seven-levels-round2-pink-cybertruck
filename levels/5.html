Level 5 of 7

AI-generated video

Generate the media you don't have. A one-prompt product film is what separates 'AI website' from 'production website'.

The technique

In OpenArt: generate product-shot candidates with an image model (Nano Banana here), pick one, then feed it to a video model (Kling/Seedance) as both the start and end frame — the rotation loops perfectly back to where it began.

Then hand Claude the file and two words of art direction: "be relevant." Both models chose sensible placement, muted autoplay, and correct aspect handling without further instruction — the cheapest, fastest level in both rounds.

The prompts that did it

Slot in the generated film
On the pink-cybertruck-web-level-4 file, please help Slot in the attached Pink Cybertruck video and be relevant. @pink-cybertruck-video-by-openart.mp4 keep pink-cybertruck-web-level-4.html untouched. Name it as pink-cybertruck-web-level-5

One prompt each. Both models reached for the same pattern: muted + playsinline autoplay gated by an IntersectionObserver — browser autoplay policy known cold, twice in a row. Opus 4.8 also generated a poster frame unprompted, the exact same flourish it added in round 1. Fable verified playback in a real browser before closing.

Cost & time

What changed vs level 4

  • Real motion media enters the page — the single biggest perceived-quality jump so far.
  • Scroll-triggered, muted autoplay: both models knew the browser policy without being told.
  • The start-frame = end-frame trick makes the loop seamless.

What this level taught

  • Media assets are now a one-prompt generation away — the leverage is in the asset, not the integration.
  • Give video models a start and end frame for loopable product shots.
  • Model habits are stable across rounds: Opus's unprompted poster frame is now a two-for-two behavior.
